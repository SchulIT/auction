# Auktionen

Mit diesem Tool können im schulischen Kontext Auktionen durchgeführt werden.

## Voraussetzungen

* Webserver
  * Apache 2.4+ oder
  * nginx
* PHP 8.4+ mit folgenden Erweiterungen
  * ctype
  * curl
  * dom
  * fileinfo
  * iconv
  * json
  * libxml
  * mbstring
  * openssl
  * pdo_mysql
  * simplexml
  * tokenizer
  * xml
  * xmlwriter
  * xsl
* MariaDB 10.4+ (ein kompatibles MySQL kann funktionieren, ist jedoch nicht getestet)
* Composer 2+
* Git
* SSH-Zugriff zur Installation

Die Software muss auf einer Subdomain betrieben werden. Das Betreiben in einem Unterverzeichnis wird nicht unterstützt.
